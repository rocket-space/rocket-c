
A simple CLI for scaffolding your projects.

### Installation

Prerequisites: [Node.js](https://nodejs.org/en/) (>=4.x, 5.x preferred) and [Git](https://git-scm.com/).

``` bash
$ npm install -g rocket-cli
```
